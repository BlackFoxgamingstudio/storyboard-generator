Storyboard Generator Documentation

Storyboard Generator is a tool to help R&D Product teams develop and create their own storyboards to help with the product design process. This project was inspired by the the CMBU UX team’s 2017 RADIO poster ("Using Storytelling to Promote Customer Experience,” selected for the People’s Choice prize). Their poster asserted the value of using storytelling as a strategic tool to be used by teams throughout the product development process to build empathy for our user’s needs and create a shared vocabulary between teams.

Story Syntax

Specify the place ("SETTING") your story takes place and the characters acting in your story.
Use descriptive words in the parentheses to add more flavor to your scene or character. Once a description is set, you don't need to declare it again until it changes.
To add a new panel change the setting; the background image the actors are placed on will change. Scenes can also be embelished with parenthetical descriptors
Signify speech with ":".
The Narrator is a special type of character and adds a caption to the panel.

#Title
SETTING (property)
Character: (property) speak
Narrator: caption text
//Comments

Sample Syntax

  // this is a sample
  #Customer Confusion
  AT COMPUTER (computer default)
  Mark (green, bronze, confused)
  Narrator: Mark is very confused
